# 10 — Future Ideas

A backlog of features to consider after the MVP is shipped. Not prioritized yet.

## Potential Features

- **Rest timer with audio/vibration** — buzz the phone when rest is up
- **Plate calculator** — given a target weight, show which plates to load on each side of a barbell
- **Bodyweight tracking** — log weight over time, factor into progress views
- **RPE / effort logging** — add a difficulty rating per set (Rate of Perceived Exertion)
- **Supersets / circuits** — group exercises that are done back-to-back
- **Export to CSV / Google Sheets** — for users who still want a spreadsheet backup
- **Dark/light mode toggle** — currently dark-only
- **Offline support / PWA** — cache API responses so the app works in a bad gym signal
- **Multiple users / sharing** — share workout templates with a friend
- **Coach mode** — create and assign programs to other users
