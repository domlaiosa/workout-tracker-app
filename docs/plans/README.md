# LiftLog — Feature Plans

This folder contains implementation plans used to drive agentic builds. Each plan is a self-contained brief for a specific feature or change.

---

## How to Use a Plan

1. Write or copy the plan template below into a new file: `docs/plans/feature-name.md`
2. Fill out each section — the more specific, the better the output
3. Hand the plan to the agent with relevant spec docs as context:
   > "Implement `docs/plans/feature-name.md`. Reference `docs/02-database-schema.md` and `docs/03-api-design.md` for existing contracts."
4. For large plans, point the agent at a specific section:
   > "Implement section 3 (Component Breakdown) of `docs/plans/feature-name.md`."
5. Once built, delete or move the plan to `docs/plans/archive/`
